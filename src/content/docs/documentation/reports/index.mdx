## ¿Por qué es importante generar informes?

Cuando trabajamos con pacientes terapeuticos, es necesario generar informes de manera trimestral, mensual o semanal. En **mindbridge** hemos creado la sección de **informes** para que puedas generar informes de tus pacientes de una manera facil y sin complicaciones.

### Generar un informe
Para iniciar el modulo de informes, debemos seleccionar **Informes** en el encabezado de la aplicación.

![Desplegable del modulo informes](/docs/reports/1.png)

Una vez dentro del modulo de informes, en el encabezado presionamos en el boton **+**.

El modulo es muy similar a la creación de evoluciones.

Los campos que podemos encontrar son:
1. **Paciente**: Nos permite seleccionar el paciente al cual le vamos a generar el informe.
2. **Descripción**: Es un campo de texto libre donde podemos escribir algun comentario que nos resulte relevante.
3. **Circulo**: Esto nos permite compartir el informe con algun circulo de colegas (opción que veremos mas adelante).
4. **Observaciones**: Es un campo de texto enriquecido donde podemos escribir el informe del paciente.
5. **Etiquetas**: Podemos agregar etiquetas a la evolución para poder filtrarlas mas adelante.

![Generar informe](/docs/reports/2.png)

Una vez completado el informe, presiona en **Crear Informe** y listo, has generado el informe de tu paciente 🎉.

![Informe del paciente](/docs/reports/3.png)

Dentro del listado, podrás ver el informe que has generado junto con las opciones permitidas para el mismo.

`Las opciones editar y eliminar están siempre habilitadas`
